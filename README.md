# convergen
A convergen ToDO Task built using the MERN stack

## :heart: Getting Started

<ol>
<li>Run Convergen App node server </li>
<li>Install  dependencies:</li>
 <code> npm install </code>
 
<li>change MONGO_URI in /server/config/keys.js
<li> run the App:</li>
 <code> npm start  </code>
</ol>
